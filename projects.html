Internship · Garofalo & Partners · Sep–Nov 2025
Integrated Industrial Monitoring System (Industry 4.0)

End-to-end system for the acquisition, storage and real-time visualization of data from industrial PLCs via the Modbus protocol. The system collects process variables (temperatures, pressures, machine states), persists them on PostgreSQL and exposes them via RESTful API and WebSocket for remote real-time monitoring.
